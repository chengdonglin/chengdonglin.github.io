本地搭建 ollama ​

概述 ​

Ollama 是一个开源框架,专为在本地机器上便捷部署和运行大型语言模型(LLM)而设计. 可以在 window, linux 上搭建，具体参考(https://ollama.com/)

docker 搭建 ​

docker 搭建参考(https://ollama.com/blog/ollama-is-now-available-as-an-official-docker-image)

可以通过 docker ps 来查看是否运行成功

这里由支持的模型列表(https://ollama.com/library)

我们采用千问模型 ollama run qwen2 , 这样就可以下载模型

python sdk 接入 ​

python
import ollama

client = ollama.Client(host="http://10.13.3.21:11434")

res = client.chat(model="qwen2", stream=False, messages=[{"role": "user", "content": "中国四大名著是什么"}],
                  options={"temperature": 0})

print(res)
